# Roadmap

## Court terme (0-3 mois)
- Ajouter RBAC et audit des modifications.
- Fournir un export CSV/JSON standardisé pour échanges SI.
- Packager la librairie Excel en local (suppression de la dépendance CDN).

## Moyen terme (3-6 mois)
- Introduire un stockage base de données (PostgreSQL) + historisation.
- Ajouter des connecteurs CMDB (GLPI/ServiceNow/NetBox).
- Améliorer la simulation d’incident (RTO/RPO, modes dégradés).

## Long terme (6-12 mois)
- Multi-tenant complet GHT avec cloisonnement.
- API publique documentée + versioning des modèles.
- Observabilité (metrics/logs) et tests de charge.
